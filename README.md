# Data_visualization projects

🔹 Overview
This project demonstrates sales data analysis using both Excel and Power BI. It includes dashboards built using both tools to compare traditional and modern data visualization techniques.

🔹 Power BI Dashboard
- KPI Cards: Sales, Profit, Orders, Profit Margin  
- Monthly Sales Trend  
- Region-wise Analysis  
- Top 5 Products  
- Interactive Slicers  

🔹 Excel Dashboard
- Built using Pivot Tables and Charts  
- Basic sales analysis  
- Category and region insights  

🔹 Tools Used
- Power BI  
- Microsoft Excel  
- DAX
